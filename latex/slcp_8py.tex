Einfache Chat-\/\+Klasse (SLCPChat) für das Chat-\/\+Projekt. 

Diese Klasse kapselt die komplette Chatlogik für einen Nutzer\+:
\begin{DoxyItemize}

\item Textnachrichten und Bilder können an bekannte Nutzer gesendet werden.
\item Eingehende Datagramme (Text) werden über einen Callback an die CLI weitergereicht. 
\end{DoxyItemize}
