Detailed Description

Diese Datei enthält Funktionen zum Senden und Empfangen von Bildern über UDP.

Sie ermöglicht das einfache Teilen von Bildern zwischen Benutzern in einem Netzwerk.

Note
Diese Datei wird in SLCPChat verwendet, um Bilder zu senden und zu empfangen.
