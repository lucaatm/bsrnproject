Function Documentation

◆ input_loop()

cli_chat.input_loop ( )

Diese Funktion führt eine Endlosschleife aus, in der der Benutzer Eingaben tätigen kann.

Sie verarbeitet die Eingaben des Benutzers, um verschiedene Chat-Kommandos auszuführen, wie JOIN, LEAVE, WHO, MSG und IMG und das Ändern der Konfiguration. Außerdem gibt es die Möglichkeit, per "HELP" eine Übersicht der verfügbaren Befehle zu erhalten.

◆ leave_chat()

cli_chat.leave_chat ( )

Diese Funktion wird aufgerufen, wenn der Benutzer den Chat verlässt.

Sie sendet eine LEAVE-Nachricht an den Discovery-Service und gibt eine Bestätigung aus. Im Falle eines Fehlers wird eine Fehlermeldung ausgegeben.

Note
Sie wird durch die atexit-Bibliothek ausgeführt, um sicherzustellen, dass sie beim Beenden des Programms aufgerufen wird.

◆ main()

cli_chat.main ( )

Die main-Funktion startet den Discovery-Service und die Chat-Funktionalität.

Sie startet Threads im "daemon"-Modus, um Nachrichten zu empfangen und Bilder zu senden, und wartet auf Benutzereingaben in einer Endlosschleife.

◆ on_message()

cli_chat.on_message ( message,
addr )

Diese Funktion verarbeitet eingehende Nachrichten und führt dann individuelle Aktionen aus.

Sie überprüft, ob die Nachricht mit "KNOWNUSERS" beginnt, um die Liste der bekannten Benutzer zu aktualisieren. Hierzu wird dann eine Tabelle mithilfe der PrettyTable-Bibliothek erstellt und ausgegeben.

Wenn die Nachricht nicht mit "KNOWNUSERS" beginnt, wird sie als normale Chat-Nachricht behandelt und ausgegeben. Dazu wird überprüft, ob der Nutzer inaktiv ist und somit eine automatische Antwort gesendet werden soll. Falls ja, wird die Abwesenheitsnachricht an den Absender zurück gesendet.

Parameters
messageDie empfangene Nachricht.
addrDie Adresse des Absenders der Nachricht.

◆ chat

cli_chat.chat = SLCPChat(HANDLE, PORT, IMAGE_PORT, WHOIS_PORT)

Initialisierung des SLCPChat-Objekts mit den geladenen Konfigurationen.

Dieses Objekt wird letztendlich verwendet, um Nachrichten zu senden, Bilder zu teilen und die Chat-Funktionalität zu verwalten.

◆ config

cli_chat.config = toml.load("resources/config.toml")

Die Konfigurationsdatei wird eingelesen, um alle wichtigen Benutzereinstellungen zu laden.

Diese Einstellungen umfassen den Benutzernamen, die Ports für Nachrichten und Bilder sowie die Whois-Portnummer.

Note
Für den Image-Port wird ein Standardwert von 6000 verwendet, falls nicht anders konfiguriert.
